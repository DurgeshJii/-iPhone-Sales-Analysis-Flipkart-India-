# -iPhone-Sales-Analysis-Flipkart-India-
This project performs an end-to-end exploratory data analysis (EDA) on iPhone sales data from Flipkart (India) to uncover pricing trends, customer behavior, discount impact, and value-for-money insights.

Using Python, Pandas, Matplotlib, and Seaborn, the analysis transforms raw e-commerce data into actionable business insights that can support pricing strategy, marketing decisions, and product positioning.
